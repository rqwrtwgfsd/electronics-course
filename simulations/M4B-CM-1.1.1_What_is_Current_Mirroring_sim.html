Current Mirror — What is Current Mirroring

Mirror Configuration

Ideal Output Current: 100.0 μA
Actual Output Current (with λ): 100.0 μA
Current Error: 0.0 %
Key Equation: I_out = I_ref × (W/L)₂/(W/L)₁ × (1 + λ·V_DS)
The ideal ratio is set by geometry. Channel length modulation introduces a V_DS-dependent error that grows with λ.

Current Bar Comparison

The sweep plot shows I_out vs V_out. Below V_DS,sat the mirror falls out of saturation and current drops. Above V_DS,sat the mirror is regulated and current is nearly flat — ideal mirroring.
