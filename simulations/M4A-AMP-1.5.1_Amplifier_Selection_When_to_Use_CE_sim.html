M4A-AMP-1.5.1 — Amplifier Selection: When to Use Common-Emitter

CE vs op-amp decision: noise figure, bandwidth, gain accuracy, input impedance — where CE wins

Noise Figure vs Collector Current (CE Stage)

NF = 1 + (Rb + re/2)/Rs + gm×Rs/(2β). There is an optimal IC that minimizes NF for a given Rs. High IC → high shot noise (last term); low IC → high Johnson noise (middle term).

Gain Accuracy: CE (without degeneration) vs Op-Amp

CE gain = gm×RC ∝ IC/T_abs. Without degeneration, gain drifts –0.3%/°C with temperature. Op-amp gain is set by resistor ratios: < 10 ppm/°C with precision parts. The gap is 300–1000×.
